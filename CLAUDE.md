# Milar Arquitetura - Tech & Style Guidelines

## Tech Stack
- Framework: Next.js 14+ (App Router)
- Styling: Tailwind CSS
- UI Components: Shadcn UI (Lucide Icons)
- Backend/Auth: Supabase
- Emails: Resend

## UI/UX & Design System
- **Vibe:** High-end architecture, minimalist, editorial, neuroarchitecture focused.
- **References:** Yellowtrace, Casa de Valentina.
- **Typography:** Elegant serif for headings (e.g., Playfair Display or similar), clean sans-serif for body (e.g., Inter or Montserrat).
- **Colors:** Neutral palette. Whites, off-whites, soft stones, deep charcoal for text. Let the project images be the primary source of color.
- **Layout:** Asymmetric grids for the portfolio, abundant whitespace, large high-quality image placeholders. 

## Development Rules
- ALWAYS use TypeScript.
- Build responsive mobile-first layouts.
- Do not use placeholder text (Lorem Ipsum) if real content is provided.
- For the Budget Simulator: Keep the form clean (multi-step if necessary) and ensure error handling is user-friendly.
